## Dirección de diseño v3

**Layla base + Thats photo headers**

- Light-first con dark mode secundario
- Fondo crema `#F5F0E8` (light) / Verde oscuro `#0D120F` (dark)
- Acento principal: Turquesa `#1B9E8F`
- Cards pasteles por tipo de sección
- Photo headers con gradiente a sangrado completo
- Tipografía: DM Sans exclusivamente
- **CERO emojis** — solo SVG Lucide

## Colores rápidos

```
Turquesa principal: #1B9E8F → #0BC5B8 → #06B6D4
Gradiente:         linear-gradient(135deg, #1B9E8F, #0BC5B8, #06B6D4)
Crema base:        #F5F0E8
Texto:             #2D1F14
Dark bg:           #0D120F
```

## Regla absoluta

> **CERO emojis en toda la UI.** Solo SVG Lucide con stroke-width 1.8, stroke-linecap round, stroke-linejoin round.
