# ermix Brand Assets v2.0

## Estructura

```
svg/                    → Vectoriales editables
png/transparent/        → PNG fondo transparente (todos los tamaños)
png/dark/               → PNG sobre fondo #09000F
png/white/              → PNG sobre fondo blanco
swatches/               → Paleta de colores SVG y PNG
```
